Vision Design System

Vision is the design system used in all Perceptyx products.

My role

Client

Perceptyx

When?

2022

The project

When facing the challenge of redesigning the entire set of platforms for Perceptyx, we decided as a team that the best way to achieve this was to create a design system.

That's how Vision was born! A design system based in Bootstrap components that is always growing and always changing.

The team

The entire team of 6 product designers worked on this project as we needed everyone's input to make sure Vision covered all the aspects needed to build every platform.

Goals

  • The company had several platforms (both internal and external) that were created across different points in time. So the main goal behind this project was to redesign all those platforms, which was going to be done by different designers, but using the same style and components in all of them.

  • The main goal was to create basic components that could be reused across all the platforms.

  • Once the basic components were created, we needed to cover the specific needs of each platform.

Users

  • When trying to define the users for a project, we always try to make it as specific as possible. However, this project involved both all diferent clients and employees.

  • The user group was composed by users from all ages, computer knowledge, professions and countries.

  • What they all had in common was a need to have an easy to use, easy to learn user experience. Some of these platforms are internal, others are self-service. Some had been in the market for years and others were new or not even launched. So the design system needed to be as simple as possible without losing the brand's personality.
