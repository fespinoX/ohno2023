Client

Trailblazer

When?

2019

The project

Traiblazer is a Ruby framework that provides a modern approach to implementing business logic.

We had a very short time to create a landing page and a documentation hub for them.

The start point for the design was the cover image that they provided to us.

The team

I worked alongside a frontend developer on this project.

Goals

  • Create a landing page with an "edgy" design using an image as a parting point for the identity.

  • Create a documentation hub that features menus and sub-menus where documentation could be uploaded and updated.

Users

  • The public for this project consisted of users with a tech-oriented profile

  • These users are familar with framework documentation, so it was important to keep a familiar structure so it was easy to navigate.

Process

I started by creating some lo-fi mock-ups to confirm the information structure and the layout with the client.

Since we had a short time to complete this project, I worked very closely with the front-end developer to ensure we could work simultaneously.

I then started creating a set of re-usable components that were going to be used in both pages.

The client wanted the pages to have personality but for it to be simple at the same time. So I created a few elements that could bring that feel from the reference image to the pages. We decided to go with a "treasure map" vibe, but without making it too obvious.

We worked on the documentation hub making sure that it could be escalated without our help as it was a framework and they always grow and need to have updates on their documentation
